## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

Library-API contains REST services to use in application demo. 

### Clone the repo

```shell
git clone https://github.com/iscrivera/library-api.git
cd library-api
```

## Postgres Database Installation

You can use the docke-compose.yml to create a database container with docker.

```bash
$ docker-compose up -d postgres
```

Run the script database/init.sql into database.


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Swager Open API

You can test the API with swagger in this url:

[http://localhost:3000/api/](http://localhost:3000/api)

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
